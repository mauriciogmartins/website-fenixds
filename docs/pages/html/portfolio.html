Sitec

Este projeto tem como alvo o gerenciamento comercial, visando vários aspectos de todos os tipos de comércio, tendo módulos opcionais para gerenciar estoque, orçamento, ordens de serviço, pedidos, para que você tenha as informações do seu negócio de forma simplificada. Sempre em desenvolvimento para melhor atender as suas necessidades.
Atualmente esta na versão 1.5 para os sistemas operacionais Windows 7/8/10.

Saged

Este projeto tem como alvo o gerenciamento educacional, de forma a organizar as informações dos estudantes e assim facilitar a instituição de ensino pode gerir seus recursos. Está em fase desenvolvimento mas ja com o módulo de cadastro de alunos liberado para uso.
Atualmente esta na versão 1.0 para os sistemas operacionais Windows 7/8/10.

Aplicativos personalizados

Desenvolvemos projetos personalizados, analisando suas necessidades e criando soluções para que você possa concentrar sua atenção aos demais aspectos do seu negócio. Por ser um projeto somente seu, o mesmo terá uma grande capacidade de adaptação e vai evoluir com você.
